5. 운영체제의 개념과 기능

  1. 운영체제는 사용자와 하드웨어 사이에서 중개자 역할을 하는 시스템 소프트웨어이다.
  2. 운영체제는 아이콘·메뉴와 같은 사용자 인터페이스를 제공하여 편리하게 컴퓨터를 사용하게 한다.
  3. 운영체제는 CPU, 메모리, 입출력장치 등의 시스템 자원을 효율적으로 관리한다.
기능 설명
사용자 인터페이스 제공 사용자가 컴퓨팅 시스템과 상호작용할 수 있도록 화면이나 메뉴 등을 이용하여 사용자 인터페이스를 제공한다.
프로세스 관리 실행 중인 여러 프로세스가 충돌 없이 실행할 수 있도록 관리한다.
주기억장치 관리 여러 개의 프로그램을 한정된 주기억장치의 크기·용량에 효율적으로 할당하고 관리한다.
입출력장치 관리 컴퓨터에 연결된 다양한 입출력장치를 제어하고 관리한다.
파일 관리 파일과 폴더를 생성·저장·삭제할 수 있도록 관리한다.

Part 2. 피지컬 컴퓨팅 기초

6. 피지컬 컴퓨팅 핵심 개념 빈칸 채우기

  1. 피지컬 컴퓨팅 : 다양한 센서로 주변의 환경 변화를 읽고, 처리 장치로 프로그램을 실행한 후, 액추에이터를 통해 출력하면서 외부 세계와 상호작용하는 컴퓨팅 기술이다.
  2. 입력장치 : 센서를 통해 주변 환경의 변화(밝기, 소리, 온도 등)를 감지·측정하여 처리 장치로 전달하는 장치이다.
  3. 처리장치 : 마이크로컨트롤러가 포함된 장치로, 센서로 인식한 데이터를 처리하고 그 결과를 출력 장치로 보내는 장치이다.
  4. 출력장치 : 컴퓨팅 시스템의 명령을 실제 물리적인 동작(빛·소리·움직임 등)으로 변환하는 장치이다.
  5. 마이크로컨트롤러 : 피지컬 컴퓨팅 시스템에서 두뇌 역할을 하는 작은 컴퓨터로, CPU·메모리·입출력 포트를 하나의 칩에 포함한다.

Part 4. 데이터의 구조화 및 분석

15. 데이터 구조화 핵심 개념 빈칸 채우기

  1. 데이터구조화 : 전달하려는 데이터의 내용 요소들을 특성에 맞게 정리·배열하여 통일된 모양으로 표현하는 것이다.
  2. 목록(리스트) : 일정한 기준에 맞추어 데이터를 순서대로 나열한 것이다. (예: 건물 층별 안내도, 시간별 일정)
  3. 표(테이블) : 서로 다른 두 기준을 세로줄과 가로줄로 나누어 데이터를 표현한 것이다. (예: 학급 시간표, 용돈 기입장)
  4. 다이어그램 : 그림·도형·점·선·화살표 등을 사용하여 데이터를 비교하거나 관계를 시각적으로 표현한 것이다. (예: 지하철 노선도, 가계도)

💡 각 구조화 종류에 알맞은 특징 연결하기

  • [1] 목록(리스트) → ③ / ① 그림·도형·점·선·화살표 등으로 데이터 사이의 관계를 그림으로 표현한 것
  • [2] 표(테이블) → ② / ② 일정한 기준에 맞추어 데이터를 순서대로 나열한 것
  • [3] 다이어그램 → ① / ③ 서로 다른 두 기준인 세로줄과 가로줄로 나누어 표 형태로 표현한 것

17. 디지털 트윈(Digital Twin) 이해하기

디지털 트윈이란?
현실 세계의 사물이나 시스템을 가상공간에 똑같이 복제하여, 시뮬레이션으로 결과를 예측·분석하는 기술이다.
활용 분야 디지털 트윈 적용 사례
제조·공장 실시간 데이터를 기반으로 가상 공장을 구축하여 생산 공정을 최적화 합니다. 장비의 고장을 사전에 예측(예측 보전)하고, 실제 라인을 가동하기 전 시뮬레이션을 통해 불량률을 최소화합니다.
스마트 시티 도시 전체를 가상 세계에 구현하여 교통 흐름 제어, 재난 시뮬레이션(침수, 화재 등), 환경 오염 모니터링 에 활용합니다. 이를 통해 도시 운영의 효율성을 높이고 시민의 안전을 확보합니다.
항공·우주 항공기 엔진이나 우주선의 가상 모델을 만들어 실시간 상태를 점검 합니다. 극한 환경에서의 기체 변화를 예측하고, 유지보수 시점을 정확히 파악하여 사고를 방지하고 비용을 절감합니다.

원칙 설명
인간의 존엄성 인공지능은 인간의 생명은 물론 정신적 및 신체적 건강에 해가 되지 않는 범위에서 개발 및 활용되어야 한다.
사회의 공공선 인공지능은 지능정보사회에서 소외되기 쉬운 사회적 약자와 취약 계층의 접근성을 보장하도록 개발 및 활용되어야 한다.
기술의 합목적성 인공지능 기술은 인류의 삶에 필요한 도구라는 목적과 의도에 부합되게 개발 및 활용되어야 하며 그 과정도 윤리적이어야 한다.

20. 사이버 폭력의 유형

유형 설명
사이버 명예 훼손 사이버 공간에서 상대방을 비하할 목적으로 사실 또는 거짓을 말하여 명예를 떨어뜨리거나 인격을 침해하는 행위이다.
사이버 스토킹 사이버 공간에서 원하지 않는 문자, 사진, 동영상을 반복적으로 보내 상대방에게 공포심이나 불안감을 주는 모든 행위이다.
사이버 갈취 및 강요 주로 와이파이 셔틀, 게임 머니 (사이버 머니) 등 사이버상 갈취 형태로 괴롭히는 행위이다.
신상 정보 유출 및 폭로 정보 통신망을 이용하여 상대방의 동의 없이 개인의 사생활과 관련된 특정 신체 부위나 각종 유해성 사진, 영상 등을 전송· 유포하여 괴롭히는 행위이다.
사이버 언어폭력 채팅방, 게시판 등 사이버 공간에서 문자, 사진, 동영상 등 으로 비방 글, 악성 댓글, 욕설 등을 올리는 행위이다.
사이버 따돌림 사이버 공간에서 특정 상대를 대화에 참여하지 못하게 하거나 채팅방에서 나가지 못하게 하는 등의 행위이다.

21. 사이버 범죄의 유형

유형 설명
해킹 불법으로 정보 시스템에 침투하여 정보를 빼내거나 프로그램을 파괴하는 행위
분산 서비스 거부 공격(DDoS) 많은 수의 컴퓨터를 원격 조종해서 특정 웹사이트에 동시에 접속하여 과부하를 일으키는 행위
악성 프로그램 컴퓨터 시스템이나 데이터를 훼손하거나 불법 활동을 수행하는 악의적인 프로그램을 배포하는 행위

💡 사이버 범죄를 예방하기 위해 우리가 실천할 수 있는 방법 (2가지)

  1. 운영체제(OS) 및 소프트웨어의 최신 보안 패치를 설치하고 버전을 업그레이드한다.
  2. 바이러스 백신을 설치하고 실시간 감시 기능을 활성화한다.

22. 개인 정보의 정의와 유형

개인 정보의 정의:
개인 정보는 살아 있는 개인에 관한 정보로 이름, 주소, 주민 등록 번호, 전화번호, 아이디와 비밀번호, 지문·홍채 정보 등 특정 개인을 드러낼 수 있는 정보이다.
유형 포함되는 정보 예시
인적 사항 성명, 주민등록번호, 주소, 연락처, 생년월일, 출생지, 성별 등
신체적 정보 얼굴, 홍채, 음성, 지문, 키, 몸무게, 건강 상태, 진료 기록 등
정신적 정보 종교, 가치관, 정당·노조 가입 여부 등
